## CSS Activation for Tabular Numbers

For proportional fonts with tabular number support, activate via CSS:
```css
font-feature-settings: 'tnum';
/* or */
font-variant-numeric: tabular-nums;
```

Fonts with confirmed tabular number support: Inter, Source Sans 3, Noto Sans, IBM Plex Sans, JetBrains Mono, Fira Code, Source Code Pro.

All monospace fonts have inherent tabular alignment by design.
